## Demo

Demo deployed at this [link](https://klimov-code.github.io/ecwid-test/)

## Local usage

Environment:

- Node.js 20
- pnpm 8

Install dependencies:

```bash
pnpm i
```

Start development server:

```bash
pnpm start
```
